# Therapii Documentation

Therapii is a Flutter and Firebase application for therapist-client engagement, guided onboarding, AI-assisted communication, journal publishing, and admin operations.

This documentation set is structured for three audiences:

- Product and operations teams who need to understand the user journeys
- Engineers who need to work in the Flutter, Firebase, and Cloud Functions codebase
- Admins who manage therapist approvals, content, billing access, and platform settings

## Stack at a glance

- Frontend: Flutter Web with Material UI
- Authentication: Firebase Authentication
- Database: Cloud Firestore
- File storage: Firebase Storage
- Backend workflows: Firebase Cloud Functions on Node.js 20
- Payments and billing: Stripe through callable Cloud Functions
- Email delivery: Resend
- AI integration: OpenAI-backed configuration through Functions and admin settings

## Core application surfaces

### Public and authentication

- Landing page and marketing surface
- Sign up and login for therapists and clients
- Email verification and role-aware post-login routing

### Client experience

- Client dashboard
- Therapist connection via invitation flow
- Human chat and AI-assisted conversation surface
- Voice conversation and support center
- Billing and plan management

### Therapist experience

- My Patients workspace
- Patient details, conversations, summaries, and voice check-ins
- Therapist approval-aware access gating
- Profile and practice setup flows

### Journal platform

- Journal portal for readers
- Favorite article saving
- Journal article detail view
- Admin content studio and analytics views

### Admin console

- Therapist approvals
- Usage and activity metrics
- Landing content configuration
- Journal administration
- User access grants tied to Stripe-related account data
