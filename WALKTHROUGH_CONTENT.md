## üí° GENERAL WALKTHROUGH FEATURES

### Design Principles:
1. **First-time only**: Each walkthrough shows once per user per page
2. **Always skippable**: Users can tap anywhere to advance or skip
3. **Persistent**: Progress saved in SharedPreferences
4. **Replay available**: Can be triggered again from Support Center
5. **Context-aware**: Only shows on relevant pages
6. **Non-intrusive**: Dark overlay with focused highlights

### Navigation:
- Tap anywhere on screen to advance to next tip
- "Skip" button available on each tooltip
- Automatic progression through sequence
- Final screen: "All set! Explore on your own now."

### Technical Implementation:
- Uses `tutorial_coach_mark` package
- GlobalKey references for each target
- SharedPreferences for persistence
- Separate walkthrough per page
- Reusable WalkthroughContent widget
